# SUYEO GIM — Portfolio

[Visit suyeogim.com](https://suyeogim.com)

The official portfolio website of fashion designer SUYEO GIM. The site presents custom garments, stage costumes, fashion research, material development, and visual direction through an editorial, image-led experience.

Each case study documents both the finished work and the process behind it, including references, textile research, pattern development, hand construction, fittings, and final performance imagery.

## Portfolio Sections

- **Works** — commissioned garments and performance projects
- **Projects** — independent fashion, textile, and visual research
- **Collection** — selected collection work
- **About / CV** — designer profile and professional background

## Experience

- Responsive editorial layouts for desktop and mobile
- Image and video galleries with animated full-screen viewing
- Project carousels and detailed process documentation
- Viewport-aware, looping video playback
- Optimized preview assets with high-quality originals retained for viewing
- Keyboard-accessible slide navigation on long-form research projects

## Technology

- React 19
- TypeScript
- Next.js-compatible Vinext runtime
- Vite
- Cloudflare-based OpenAI Sites hosting

## Local Development

Node.js 22.13 or later is required.

```bash
npm install
npm run dev
```

Create a production build:

```bash
npm run build
```

Run the project validation suite:

```bash
npm test
```
